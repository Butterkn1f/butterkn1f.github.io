Operation I.C.C.

3D Game made in OpenGL for Studio Project.

  • Team of 4

  • February 2022

  • C++ / Visual Studio / OpenGL

  • Source code currently hidden.

The theme we were given for the project was Scam. Our team decided to make an FPS detective game to catch the scammers.

I was in charge of making the cutscenes, sound, interactions, environment, and lighting.

The greatest challenge I faced while making this game was figuring out how to do the interactions. Essentially, I want items to be highlighted when the middle of the camera is pointing at the object in a 3D space.

In the end, I figured it out by making use of raycasting. I projected a 3D ray from the mouse. The mouse position is in 2D coordinates, so I then transform the coordinates by working backwards through the 3D transformation pipeline - from Viewport Coordinates to World Coordinates. After getting the coordinates, I check whether this invisible ray intersects with objects in my scene by using Oriented Bounding Box (OBB) Collision Detection.

Then, the objects are then highlighted using modifications to the OpenGL shader, and when the user clicks, it will fire the scripted events depending on the object clicked - such as moving a picture frame, opening a drawer, or opening a book.
